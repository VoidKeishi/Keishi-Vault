# Overview
- Logic: Ngôn ngữ hình thức cho phép biểu diễn thông tin dưới dạng các kết luận có thể được đưa ra
	- Logic = Syntax + Semantics
- Syntax - Cú pháp: Xác định các mệnh đề trong một ngôn ngữ
	- Cú pháp = Ngôn ngữ + Lý thuyết chứng minh
- Semantic: Xác định ý nghĩa của các mệnh đề trong một ngôn ngữ
- Tính chất
	- Tính bao hàm
	- Tính đúng đắn
	- Tính hoàn chỉnh
